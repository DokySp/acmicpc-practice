### 메모
 - 단순한 덱(Deque) 구현 문제
 - **덱(Deque)**이란?
    - 큐와 스택의 장점만을 가져다 만든 자료구조
    - 큐와 스택은 **말단에 있는 자료를 가져다 쓰는 자료구조**이고, 덱은 양쪽 끝에 저장되어 있는 값은 열람이 가능하다!
    - 단, **중간에 있는 값을 열람**하는 것은 자료구조의 특성상 **불가능**하다!

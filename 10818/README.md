 - 최댓값, 최솟값 문제 초기값 주의점!
    - max에는 최솟값, min에는 최댓값을 주어야 비교 가능! 이 때,
       1. 문제 조건의 최대, 최소 범위를 초과한 값을 미리 할당.
       2. 맨 처음 사용자 입력값을 min, max에 할당.
    - 위에 두 방법 중, 하나를 택해서 문제 풀 것! (조건이 int범위 내일 경우, 1번 방법 추천)
